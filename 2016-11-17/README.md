これは'oppai' -> 'pqqso' と変換しなさいという意味である
ある文字列(s)を受け取ったときにこの変換をして出力してください

```
a -> s
b -> n
c -> v
d -> f
e -> r
f -> g
g -> h
h -> j
i -> o
j -> k
k -> l
l -> a
n -> m
m -> z
o -> p
p -> q
q -> w
r -> t
s -> d
t -> y
u -> i
v -> b
w -> e
x -> c
y -> u
z -> x
```

入力フォーマット

```
s
```

入力例

```
cat
```

出力例

```
vsy
```

入力例

```
trump
```

